# MacOSX 安装 manim 动画制作

2020年04月20日 - YuweiYin

个人基于 manim 制作的动画代码托管于 Github

  • 参考资料：
    • manim
    • manim documentation
    • manim tutorial

# 1. 在桌面（或者其它位置）创建安装目录，并克隆源码

$ cd ~/Desktop/
$ mkdir animation
$ cd animation
$ git clone https://github.com/3b1b/manim.git

# 2. conda 创建并激活虚拟环境

（Anaconda3 或 Miniconda3 安装略） （此处命名为 manim，可自定义）

$ conda create -n manim python=3.7
$ source activate manim

# 3. 确认 Python 版本

（manim 需要 Python 版本 3.7）

$ python -V

Python 3.7.6

# 4. 安装必要的环境

（此后的命令中，python 即为 python3、pip 即为 pip3） （可通过命令 $ pip list 和 $ conda list 查看已安装的软件包）

$ pip install sox
$ pip install ffmpeg
$ pip install latex
$ pip install pycairo
$ pip install -r requirements.txt

# 5. 运行测试程序

（SquareToCircle是指让测试程序产生动画：方块渐变为圆形）

$ python manim.py example_scenes.py SquareToCircle -pl
或者
$ python -m manim example_scenes.py SquareToCircle -pl

Media will be written to ./media/. You can change this behavior with the --media_dir flag.
File ready at ./media/videos/example_scenes/480p15/SquareToCircle.mp4

生成的动画位于./media/videos/example_scenes/480p15/目录下，名为 SquareToCircle.mp4

# 6. 注意 TeX 配置

全局常量文件 manimlib/constants.py 中的 TEX_USE_CTEX 表示是否使用 CTeX

TEX_USE_CTEX 对 manimlib/utils/tex_file_writing.py 文件中的 tex_to_dvi(tex_file) 函数有影响

TEX_USE_CTEX = False 时使用命令 latex -interaction=batchmode -halt-on-error -output-directory="./media/TeX" "./media/Tex/???.tex" > /dev/null

TEX_USE_CTEX = True 时使用命令 xelatex -no-pdf -interaction=batchmode -halt-on-error -output-directory="./media/TeX" "./media/Tex/???.tex" > /dev/null

当 latex 或 xelatex 出错时，返回值 exit_code 不等于 0，从而会报错。
