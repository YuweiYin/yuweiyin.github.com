# MacOSX 搭建本地 Apache 网络服务器

2020年04月18日 - YuweiYin

# 1. 创建网站目录

如果要把网站建在个人 home 目录（即 /Users/username）的 Sites 目录下，则先增加 /Users/username/Sites 目录，并把网站文件（含 index.html 或 index.php 文件等）放入其中。

# 2. 修改 httpd 配置文件

进入 apache2 目录：$ cd /etc/apache2/

备份 httpd 配置文件 httpd.conf：$ sudo cp httpd.conf httpd.conf.bak

修改配置文件：$ sudo vim httpd.conf

# 2.1. 修改服务器根目录 ServerRoot，注意目录最末尾不加斜线

默认为：ServerRoot "/usr"
修改为：ServerRoot "/Users/username"

# 2.2. 修改文件根目录 DocumentRoot

默认为：
DocumentRoot "/Library/WebServer/Documents"
<Directory "/Library/WebServer/Documents">
修改为：
DocumentRoot "/Users/username/Sites"
<Directory "/Users/username/Sites">

# 2.3. 在标签内部，修改 Options

默认为：Options FollowSymLinks Multiviews
修改为：Options Indexes FollowSymLinks Multiviews

# 2.4. 去掉一行的注释，使 PHP 文件生效

默认为：#LoadModule php7_module libexec/apache2/libphp7.so
修改为：LoadModule php7_module libexec/apache2/libphp7.so

# 2.5. 修改 ServerName 为本地 IP : 80 端口

默认为：#ServerName www.example.com:80
修改为：ServerName 127.0.0.1:80
或者是：ServerName localhost:80

保存并退出

# 3. 复制 libexec 文件夹

由于修改了服务器目录 ServerRoot，因此原目录下的 libexec 文件夹需要复制到新服务器目录下（配置文件 httpd.conf 中说明了需要 libexec 文件夹下的各种 .so 文件）：

$ sudo cp -r /usr/libexec /Users/username/

# 4. 重启Apache服务器：

$ sudo apachectl -k restart

# 5. 在浏览器中输入本地 IP : 80 端口，查看 Apache 服务器是否正常启动

  • 浏览器输入：http://127.0.0.1:80
  • 或 者 输入：http://localhost:80
