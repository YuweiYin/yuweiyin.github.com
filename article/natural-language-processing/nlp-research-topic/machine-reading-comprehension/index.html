# 2. MRC Evaluation

# 2.1. MRC Answer Type

  • Multiple Choice Question: Given several choices, the model needs to choose the correct one.
  • Cloze Test: Given a destructed sentence, in which some words are deleted, the model needs to fill the blanks with correct words.
  • Span Answer: Given a text passage, the answer of which is a substring (span), the model needs to answer the correct start and end positions (indices) of the correct span.
  • Free Answer: Since the answer format is not stipulated, the model needs to generate a sentence to answer the question.
  • Unanswerable: The question can not be answered based on the current passages. So the model needs to answer "Unanswerable".

  • Multiple Choice Question and Cloze Test belong to objective questions. When it comes to evaluation, compare sample answer with the model output and use accuracy or precision as the metric.

  • Span Answer is a kind of semi-objective question. When it comes to evaluation, compare the sample answer with the model output.

    • Exact Match: The output start and end positions (indices) must be exactly the same as those of the sample answer. Otherwise, the current question will not score.
    • F1 Score: The harmonic mean of precision and recall, formularized as F_1 = (2 * p * r) / (p + r). Plus, Weighed F1 Score: F_1_w = (2 * p * r) / (A * p + B * r), where A and B are hyper-parameter to control the preference for either precision or recall. if A = B = 1, weighed F1 Score is standard F1 Score.

  • Free Answer is the most flexible type. The most ideal metric for Free Answer is: the closer the semantic meaning between the sample answer and the model output, the higher score will the model obtain. However, semantic similarity is a core (maybe ultimate) problem in the NLP field. Human evaluation is costly, less efficient, and sometimes tricky because the evaluating standard varies from different people. Therefore, automatic evaluation metrics are widely used, such as ROUGE, BLEU, and METEOR.

  • Generally, Human Level means the score of a single human on the dataset; Full Marks means the score of several humans that cooperatively solve problems of the dataset.

# 3.2. Multi-Paragraph Dataset

MS MARCO

  • Info
    • Year: 2016
    • Institute: Microsoft Corporation
    • Language: English
    • Dataset Type: Multi-Paragraph Dataset
    • Data Source: user queries and the top results on Bing search engine
    • Dataset Size: more than 8,000,000 passages and more than 1,000,000 questions
    • Question Type: Free Answer
    • Other: MS MARCO hosts 3 competitions:
      • (1) Judge if the answer can be derived from given paragraphs. (Yes or No Question)
      • (2) Generate answer sentences. (Free Answer)
      • (3) Sort several paragraphs according to the semantic similarity with the given question.
