5.1 计算机网络的概念

# 从智能手机谈起

  • 手机里的网络术语：

    • GPRS，3G，4G
    • Wi-Fi
    • 蓝牙
    • 虚拟专用网络Virtual Private Network
    • 移动网络
    • 服务器、端口、代理、身份验证…

  • 3G

    • 3G (3rd-Generation) 是第三代移动通信技术，其规范名称为 IMT-2000（International Mobile Telecommunications-2000），是支持高速数据传输的蜂窝网络移动电话技术。
    • 3G 服务能够同时发送声音（通话）及信息（电子邮件、即时通信等）。
    • 3G 的代表特征是提供高速数据业务，速率一般在几百 kbps 以上。
    • 3G 是指将无线通信与国际互联网等多媒体通信结合的一代移动通信系统。
    • 3G 存在三种标准：CDMA2000、WCDMA、TD-SCDMA。

  • 4G

    • 4G 集 3G 与 WLAN (Wireless Local Area Network) 于一体，能够快速传输数据，包括高质量音频、视频和图像等信息。
    • 4G 能够以 100Mbps 以上的速度下载，比 ADSL (4Mbps) 家用宽带快 25 倍，并能够满足大多用户对于无线服务的要求。
    • 此外，4G 可以在 DSL 和有线电视调制解调器没有覆盖的地方部署，然后再扩展到整个地区。

  • 5G

# 计算机联网的作用

  • 共享设备 (如打印机、传真机)
  • 分布式计算
  • 云计算
  • 云存储
  • 网站平台
  • 文件传输
  • 邮件通信
  • 即时通讯
  • 视频聊天
  • 在线会议
  • 网络游戏
  • ...

# 计算机怎样联网

  • 小型的局域网 (以太网) LAN (Local Area Network)
    • 网线、网线接口、网卡、集线器
    • 软件

  • 远程网络
    • 局域网 + 路由器 (借助电话线 或 网络运营商的公网)
    • 软件

  • 家庭上网

5.2 数据通信的概念与原理

# 通信信道 (链路)

  • 什么是通信信道？
    • 物理通路或无线信号传输的频段
  • 信道中传输的是什么信号？ -电磁信号 (电缆或空气中电子的运动)
  • 什么是数字信号？
    • 按一定的频率传送的电平信号
  • 什么是模拟信号？
    • 特定频率范围内的任意值
  • 数字信号的优点：抗干扰，处理方便

# 网络节点和服务器

  • 什么是网络节点？
    • 网络中的设备
    • 计算机网络可以看成是节点和连线构成的图
  • 什么是服务器？
    • 网络中提供服务的计算机
  • 什么是客户端？
    • 网络中享受服务的计算机
  • 有些网络的服务器也是客户端，二者只是逻辑上的分离，但实质上都可以是同样的计算机机器。

# 网络拓扑结构

# 怎样找到对方 (寻址)

  • 通过地址
  • 地址的分层
    • (应用层) Domain Name 域名：www.google.com
    • (网络层) IP（Internet Protocol）地址：172.217.27.142
    • (链路层) Mac 地址：AB-CD-EF-01-23-45
  • 分层的意义：
    • 高层的地址 (域名) 比较容易让人记住
    • 中层的地址 (IP) 方便路由器快速识别网段、进行转发
    • 底层的地址 (Mac 地址) 是各机器的固定值、因此便于快速识别。
  • 什么是端口 Port？
    • 网络地址中为不同的服务开设的数据通道 (更精确的地址)，例如 172.217.27.142:443
      • 这里 443 端口即网页浏览端口，主要是用于 HTTPS 服务，是提供加密和通过安全端口传输的另一种 HTTP 协议。

# 域名管理

  • 域名服务器 DNS (Domain Name Server)
    • 使用时往往需要地址映射，利用多级的域名服务器 DNS (Domain Name Server) 进行解析
    • 全世界只有 13 台 IPv4 根域名服务器。1 台主根服务器在美国，其余 12 台均为辅根服务器
      • 辅根服务器中 9 台在美国。欧洲有 2 台：分别位于英国和瑞典。亚洲仅 1 台：位于日本。
  • 多级域名
    • 以 www.university.edu.cn 为例，顶级域名为 cn 表示中国，次级域名为 edu 表示教育机构，再往下一级的域名 university 是某学校的名称，最低级的域名是 www 表示这是该学校的 World Wide Web 万维网服务
    • 目前越来越多的企业网站不再使用 www 域名，因为有多种网络服务可以提供、可以把任务分布到多台服务器上。因此使用或不使用 www 域名往往都会定位到同一服务器，但会有 DNS 解析上的区别。

# 数据传送的方式

  • 包交换 (分组交换)、存储转发
    • 将消息分成多个数据包，可在不同信道上传送，以便共享信道
    • 区别于线路交换
    • 数据在信道中可能会通过多个节点，每个经过的节点都会先把包缓存起来，检查确定没有错误后，找到空闲的信道发送出去
  • 一个节点发往同一个目的地的数据包可能会经过不同的信道
  • 接收端需要把接收到的包按发送的时间先后还原完整的消息

# 通信协议

  • 什么是协议？
    • 从一个网络节点向另一个网络节点传送数据的约定。
  • 举例：以太网的协议
    • 发送数据的节点把数据加上地址 (打包) 发送到网上；
    • 网上的所有节点都会收到这个数据包；把包拆开，看看是否是自己的包。是则留下，不是则丢弃。

# 常见的网络设备

网卡：计算机和网络连接的设备，负责数据的收发。

集线器：将网络节点连接起来的设备，将一个节点发来的数据，广播给其他节点。

交换机：将网络节点连接起来的设备，将一个节点发来的数据，发送到指定的目的节点。

路由器：将一个网络的数据发送给另一个网络的设备。

5.3 互联网的概念

# 互联网

  • 什么是互联网？
    • 用一种协议（TCP/IP）将全世界的各种网络连接起来的网络（网络的网络）
  • 目前有超过 5 亿个节点，大于 15 亿名用户
  • 大量应用建立在互联网上

# 互联网的结构

# 即时通讯

  • 什么是即时通讯？
    • 是一种基于互联网的即时交流消息的业务。
    • 与电子邮件不同，电子邮件处理的消息不一定是实时的。
  • 允许两人或多人使用网络实时的传递文字信息、文件、语音与视频交流。
  • 常见的即时通讯服务例如 Facebook、Line、Windows Live Messenger、AOL Instant Messenger、skype、WhatsApp、Yahoo! Messenger、.NET Messenger Service、Jabber、ICQ 与 QQ 和 WeChat 等。

# 社交网络

  • 什么是社交网络？
    • 基于互联网，为用户提供各种联系、交流的交互通路。
  • 知名的社交网络服务包括：Google+、Myspace、Twitter、Facebook、微博、微信(朋友圈)、QQ(空间)、贴吧、人人网等。
    • 主要功能：
      • 记录个人数据
      • 私讯
      • 用户相互链接
      • 用户检索
      • 日记/博客 (含 图文视频)
      • 社区

# 物联网

  • 物联网
    • 基于互联网、传统电信网等信息承载体，让所有能够被独立寻址的普通物理对象实现互联互通的网络
    • 采用类似互联网的技术。
  • 物联网的作用：
    • 可以应用电子标签将真实的物体上网联结，在物联网上都可以查找出它们的具体位置。
    • 通过物联网可以用中心计算机对机器、设备、人员进行集中管理、控制，也可以对家庭设备、汽车进行遥控，以及搜寻位置、防止物品被盗等各种应用。

5.4 保障信息安全的方法

# 网络不总是安全的

  • 不安全的因素
    • 未经授权的访问，非法入侵、窃取机密
    • 发送病毒或木马，病毒使计算机功能崩溃、木马使个人隐私被泄露
    • 冒充别人的地址/身份/签名，冒名进行网络犯罪、栽赃嫁祸
    • 截获传输的数据 (MiM 中间人攻击)
    • 密集攻击/访问某个节点，使其基本服务功能瘫痪 (DoS, DDoS 拒绝服务攻击)

# 回到智能手机

  • 手机里的网络：
    • GPRS：通用无线分组业务数据通讯
    • 3G, 4G, 5G：更高效的无线分组业务数据通讯
    • Wi-Fi：无线以太网
    • 蓝牙：使用 2.4GHz 频段的近距离无线网络
    • 虚拟专用网络 (VPN)：经过加密的网络
    • 移动网络：网络运营商提供的互联网接入服务
    • 身份认证：确认合法的网络用户
