3.1 感知、控制与反馈原理

# 从无人驾驶汽车谈起

  • 问题：怎样驾驶汽车？
    • 知道行走路线
    • 知道目前的位置和方向
    • 知道速度
    • 知道路上是否有障碍物
    • 知道汽车的状况
    • 知道如何操纵汽车（方向盘、油门、刹车、灯光）
    • ...

# 信息与感知

  • 什么是信息？
    • 对人类有用的、可理解的消息
  • 问题：信息种类有哪些？
    • 文字、图像、影像、声音、温度、湿度、压力、味道...
  • 什么是感知？
    • 信息的获取

# 人类感知信息的渠道

  • 眼睛（视觉信息）：75%
  • 耳朵（听觉信息）：10%
  • 皮肤（触觉）：5%
  • 鼻子（嗅觉）：5%
  • 舌头（味觉）及其他：5%
  • 第六感觉？

# 计算机感知信息的渠道

  • 什么是传感器？
    • 可以感知光、声波、速度、方向、温度、湿度、压力、气味等物理量的设备。
  • 传感器一般把物理量转换为电信号，便于计算机感知。
  • 传感器的种类：
    • 光电传感器
    • 拾音器
    • 测速器
    • GPS
    • 温/湿度传感器
    • 倾斜传感器
    • ...

# 无人驾驶汽车需要什么传感器？

  • 知道目前的位置和方向—— GPS
  • 知道速度——测速表
  • 知道路上是否有障碍物——摄像头 + 影像分析程序
  • 知道汽车的状况——温度、油量、转速...传感器

# 计算机如何识别障碍物

  • 什么是模式识别？
    • 对表征事物或现象的各种形式的 (数值的、文字的和逻辑关系的) 信息进行处理和分析，以对事物或现象进行描述、辨认、分类和解释的过程，是信息科学和人工智能的重要组成部分。
  • 举例：
    • 从汽车摄像头中分辨道路、行人、障碍物...

# 计算机怎样把传感器测到的电信号转变成数据？

  • 一些传感器获得的电信号是模拟量，需要用模数（AD）转换器，将其转换成数字信号。
  • 模数转换是通过采样实现的。

# 控制

  • 什么是控制？
    • 为达到规定的目标，对设备或系统的工作特性所进行的调节或操作。
  • 计算机如何实现控制？
    • 计算机查询接口得到的数据，由计算机程序计算后，决定输出什么样的控制数据。
    • 计算机产生的控制数据由计算机接口发送给控制器，控制器调节相应的设备或系统的工作状态（例如步进电机的角位移或线位移）。

# 反馈

  • 什么是反馈？
    • 将系统的输出返回到输入端并以某种方式改变输入，进而影响系统功能的过程。
    • 即：将输出量通过恰当的检测装置返回到输入端并与输入量进行比较的过程。
    • 计算机不断把前一次的结果作为后一次计算的输入，影响后续的动作。

3.2 计算机接口的概念

# 传感器的数据如何送给计算机？

  • 通过计算机与外界的接口实现的。
  • 什么是计算机接口？
    • 完成计算机主机系统与外部设备之间信息交换的设施。一般由接口电路、连接器（连接电缆）和接口软件（程序）组成。
  • 接口的类型：
    • 外部：串口（RS232，USB），并口（IEEE 1284），无线接口（蓝牙），网络接口（RJ45）
    • 内部：IDE，SCSI，SATA 等

3.4 计算机系统的概念

# 通用计算机系统

  • 什么是计算机系统？
    • 软硬件协调配合的整体。

# 专用计算机系统

  • 什么是专用计算机系统？
    • 专为解决某一特定问题而设计制造的电子计算机。
    • 一般拥有固定的存储程序。

3.5 单片机与嵌入式系统的概念

# 单片机/微处理器

  • 什么是单片微型计算机（单片机）？
    • 制作在一块集成电路（芯片）上的计算机，简称单片机或微控制器。
    • 它包括中央处理器（CPU），数据存储器、程序存储器，定时/计数器，各种输入/输出（I/O）接口和时钟电路。
    • 可独立地进行工作。
    • 应用广泛，例如：
      • 电机控制、条码阅读器/扫描器、消费类电子、游戏设备、电话、HVAC、
      • 楼宇安全与门禁控制、工业控制与自动化和白色家电（洗衣机、微波炉）等。

# 嵌入式系统

  • 什么是嵌入式系统？
    • 嵌入到对象体系中的专用计算机系统。
  • 嵌入式系统的三个基本要素
    • 嵌入性
    • 专用性
    • 计算机系统
  • 对象系统是指嵌入式系统所嵌入的宿主系统。
